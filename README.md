# student-job-finder
Student Job Finder là ứng dụng web giúp sinh viên tìm kiếm việc làm part-time và thực tập một cách nhanh chóng. Hệ thống cung cấp chức năng tìm kiếm, xem danh sách công việc và xem chi tiết thông tin tuyển dụng. Dự án được xây dựng bằng Django với mục tiêu hỗ trợ sinh viên tiếp cận cơ hội việc làm dễ dàng hơn.
